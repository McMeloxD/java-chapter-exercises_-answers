题13

```
2019年1月1日起，国家推出新的个人所得税政策，起征点上调值5000元。也就是说税前工资扣除三险一金（三险一金数额假设是税前工资的10%）后如果不足5000元，则不交税。如果大于5000元，那么大于5000元的部分按梯度交税，具体梯度比例如下：
	0 ~ 3000元的部分，交税3%			
	3000 ~ 12000元的部分，交税10%
	12000 ~ 25000的部分 ， 交税20%		
	25000 ~ 35000的部分，交税25%
	35000 ~ 55000的部分，交税30%		
	55000 ~ 80000的部分，交税35%
	超过80000的部分，交税45%

比如：你入职一家企业后，税前工资是15000，则你每月该交个税的部分是15000-1500-5000=8500元，个税缴纳数额是3000×3%+5500×10%=640元。税后工资12860元。

请完成一个个税计算程序，在用户输入税前工资后，计算出他对应的纳税数额，以及税后工资为多少？
```

```java
    public static void t13() {
        double money = 15000;// 起始工资
        double wxyj = money * 0.1; // 五险一金
        double wxyjHou = money - wxyj;// 扣除五险一金后
        double shui = 0;
        if (wxyjHou >= 5000) { // 大于起征点的,要交税
            double shuiqian = wxyjHou - 5000; // 要交税的工资
            if (shuiqian > 0 && shuiqian <= 3000 ) {
                shui = shuiqian * 0.03;
            } else if (shuiqian > 3000 && shuiqian <= 12000) {
                // 3000以内的按3%扣税,超出3000部分的按照10%扣税
                shui = 3000 * 0.03 + (shuiqian - 3000) * 0.1;
            } else if (shuiqian > 12000 && shuiqian <= 25000) {
                // 3000以内的按3%扣税,超出3000小于12000的9000块钱按照10%,超过12000部分按照20%扣税
                shui = 3000 * 0.03 + 9000 * 0.1 +  (shuiqian - 12000) * 0.2;
            }else if (shuiqian>25000 && shuiqian<=35000){
                shui = 3000*0.03 + 9000*0.1 + 13000*0.2 + (shuiqian-25000)*0.25 ;
            }else if (shuiqian>35000 && shuiqian<=55000){
                shui = 3000*0.03 +9000*0.1+ 13000*0.2 +10000*0.25 +(shuiqian-35000)*0.3;
            }else if (shuiqian>55000 && shuiqian<=80000){
                shui = 3000*0.03 +9000*0.1+ 13000*0.2 +10000*0.25 + 20000*0.3 +(shuiqian-55000)*0.35 ;
            }else {
                shui = 3000*0.03 +9000*0.1+ 13000*0.2 +10000*0.25 + 20000*0.3 +25000*0.35 + (shuiqian-80000)*0.45;
            }
        }
        System.out.println("应纳税额是:" + shui );
        System.out.println("税后工资是: " + (wxyjHou - shui) );
    }
```

题17

```
已知2023年是兔年，请在控制台输出从1949年到2023年中所有是兔年的年份(考虑1-12月份整年,不考虑1月份是虎年的情况)
```

```java
    // 已知2023年是兔年，请在控制台输出从1949年到2023年中所有是兔年的年份(考虑1-12月份整年,不考虑1月份是虎年的情况)
    public static void t17() {
        // 循环开始是1949 结束是2023
        for (int i = 1949; i <= 2023; i++) {
            //如果年份和2023年的差值是12的倍数 则说明是兔年

                System.out.println(i);
            }
        }
    }
```

题目21

```
设计一个方法，删除指定下标的元素，要求从删除位置开始，之后的元素整体前移
	[1,2,3,4,5,6,7] =>删除3后 => [1,2,4,5,6,7]
```

```java
public static void main(String[] args) {
        int[] arr = {1, 2, 3, 4, 5, 6, 7};
        int[] arr2 = t28(arr, 3);
        System.out.println(Arrays.toString(arr2));
    }

    // 设计一个方法，删除指定下标的元素，要求从删除位置开始，之后的元素整体前移
    // 	[1,2,3,4,5,6,7] =>删除3后 => [1,2,4,5,6,7]
    public static int[] t28(int[] arr, int e) {
        boolean flag = false;// 假设不存在要删除的元素
        int index = -1; // 记录一下出现该元素的下标
        for (int i = 0; i < arr.length; i++) {
            if (e == arr[i]) {
                flag = true;
                index = i;
            }
        }
        if (!flag) {
            System.out.println("数组中不存在该元素");
            return arr;// 返回原数组
        }
        int[] arr2 = new int[arr.length - 1];// 存储删除后的元素
        int index2 = 0;
        for (int i = 0; i < arr.length; i++) {
            if (i == index){
                continue;
            }
            arr2[index2] = arr[i];
            index2++;
        }
        return arr2;
    }
```
